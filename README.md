## Запуск Moodle

### Команда запуска

В терминале перейдите в папку проекта и выполните:

```bash
docker-compose up -d

Убедитесь, что оба контейнера запущены:
docker ps

Откройте браузер и перейдите по адресу:
http://localhost:8080

Данные для входа

Поле	Значение
Логин	admin
Пароль	(пароль, указанный в docker-compose.yml)
