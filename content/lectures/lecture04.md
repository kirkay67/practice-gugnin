Lecture 04 — «Изображения и таблицы»
Тема: Вставка изображений и создание таблиц
Цель урока: Научиться добавлять на страницу картинки и создавать таблицы для структурирования данных.
Время: ~20 мин.
Ключевые понятия: img, src, alt, table, tr, td, th

1) Введение
Изображения делают страницы визуально привлекательными, а таблицы помогают представлять данные в удобном формате (расписания, цены, характеристики).

2) Основная часть
Изображения — тег <img> (одиночный, без закрывающего):

html
<img src="cat.jpg" alt="Рыжий кот">
src — путь к файлу изображения

alt — альтернативный текст (если картинка не загрузится или для незрячих пользователей)

Таблицы создаются несколькими тегами:

<table> — контейнер таблицы

<tr> — строка таблицы

<td> — ячейка с данными

<th> — ячейка-заголовок (жирный текст)

Пример простой таблицы:

html
<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Анна</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
</table>
(Атрибут border="1" добавляет рамки для наглядности)

3) Пример/практика
Создайте страницу "Расписание уроков". Используйте таблицу с днями недели и временем занятий. Добавьте сверху картинку с расписанием (любое изображение).

Задание:

Найдите любое изображение в интернете и сохраните в папку с вашим проектом

Вставьте его на страницу с подходящим alt текстом

Создайте таблицу 3x3 с любыми данными (например, список фильмов с годами выпуска)
