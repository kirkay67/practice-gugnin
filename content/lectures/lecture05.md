Lecture 05 — «Формы и ввод данных»
Тема: Создание форм для сбора информации
Цель урока: Научиться создавать формы с различными полями ввода.
Время: ~20 мин.
Ключевые понятия: form, input, type, textarea, button, select, option, label

1) Введение
Формы используются везде: регистрация, вход на сайт, поиск, комментарии, заказ товаров. Они позволяют пользователям отправлять данные на сервер.

2) Основная часть
Форма создается тегом <form>:

html
<form action="/submit" method="post">
  <!-- поля формы -->
</form>
action — адрес, куда отправляются данные

method — способ отправки (get или post)

Основные типы полей ввода (<input>):

html
<!-- Текстовое поле -->
<input type="text" name="username" placeholder="Введите имя">

<!-- Поле для пароля -->
<input type="password" name="password" placeholder="Пароль">

<!-- Email -->
<input type="email" name="email" placeholder="your@email.com">

<!-- Флажок (чекбокс) -->
<input type="checkbox" name="agree" id="agree">
<label for="agree">Согласен с условиями</label>

<!-- Кнопка отправки -->
<input type="submit" value="Отправить">
Многострочное текстовое поле (<textarea>):

html
<textarea name="comment" rows="5" cols="40"></textarea>
Выпадающий список (<select>):

html
<select name="city">
  <option value="msk">Москва</option>
  <option value="spb">Санкт-Петербург</option>
</select>
Важно: тег <label> связывает текст с полем ввода. При клике на текст фокус переходит в поле.

3) Пример/практика
Создайте форму регистрации с полями: имя (текст), email, пароль, чекбокс "Получать новости" и кнопка "Зарегистрироваться". Добавьте выпадающий список для выбора страны.

Задание:

Создайте форму с 3-4 различными типами полей

Используйте <label> для каждого поля

Добавьте кнопку отправки формы
