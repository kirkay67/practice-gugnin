Lecture 02 — «Работа с текстом (заголовки, параграфы)»
Тема: Форматирование текста в HTML
Цель урока: Научиться использовать заголовки, параграфы и элементы форматирования текста.
Время: ~20 мин.
Ключевые понятия: заголовок, параграф, strong, em, перенос строки

1) Введение
Текст составляет основу большинства веб-страниц. В HTML есть специальные теги для структурирования текста: заголовки разных уровней, абзацы, выделение важных фрагментов и переносы строк.

2) Основная часть
Заголовки создаются тегами от <h1> (самый важный) до <h6> (самый мелкий):

html
<h1>Главный заголовок страницы</h1>
<h2>Заголовок раздела</h2>
<h3>Подраздел</h3>
Параграфы создаются тегом <p>:

html
<p>Это первый абзац текста. Он отделяется от других абзацев отступами.</p>
<p>Это второй абзац текста.</p>
Форматирование текста:

<strong> — жирное выделение (важный текст)

<em> — курсив (логическое ударение)

<br> — перенос строки (одиночный тег, без закрывающего)

Пример:

html
<p><strong>Важно!</strong> Это <em>очень важный</em> текст.<br>Новая строка.</p>
3) Пример/практика
Создайте страницу с заголовком "Моя автобиография". Используйте <h1> для имени, <h2> для разделов "Детство", "Учеба", "Хобби". Под каждым заголовком добавьте по абзацу текста. Выделите ключевые слова жирным или курсивом.

Задание:

Создайте структуру с заголовками разных уровней

Добавьте 3-4 параграфа

Используйте <strong> и <em> для выделения текста
