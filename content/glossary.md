# Глоссарий терминов HTML

## A
**Атрибут** — дополнительное свойство тега, которое уточняет его поведение или внешний вид (например, `href` для ссылки, `src` для изображения). Записывается в открывающем теге: `<a href="url">`.

## B
**Браузер** — программа для просмотра веб-страниц (Chrome, Firefox, Safari). Браузер читает HTML-код и отображает его в понятном для человека виде.

## В
**Валидация** — проверка HTML-кода на соответствие стандартам W3C. Ошибки валидации могут привести к некорректному отображению страницы.

## Г
**Гиперссылка** — элемент страницы (текст или изображение), при клике на который происходит переход на другую страницу или другой раздел текущей страницы. Создается тегом `<a>`.

## Д
**DOCTYPE** — декларация в начале HTML-документа, которая сообщает браузеру, какую версию HTML использовать. Для HTML5 пишется `<!DOCTYPE html>`.

## З
**Заголовок** — элемент, обозначающий заголовок раздела. Бывает 6 уровней: от `<h1>` (самый важный) до `<h6>` (самый мелкий).

## И
**Изображение** — графический элемент на странице. Вставляется тегом `<img>` с обязательным атрибутом `src` (путь к файлу) и рекомендованным `alt` (альтернативный текст).

## К
**Код** — текст, написанный на языке HTML и интерпретируемый браузером для отображения веб-страницы.

## Л
**Логическая разметка** — подход, при котором теги описывают смысл содержимого (заголовок, параграф, важный текст), а не его внешний вид (жирный, красный). Внешний вид задается CSS.

## М
**Маркированный список** — список, пункты которого обозначаются маркерами (обычно точками). Создается тегом `<ul>` (unordered list).

## Н
**Нумерованный список** — список, пункты которого нумеруются по порядку. Создается тегом `<ol>` (ordered list).

## О
**Открывающий тег** — часть элемента, обозначающая начало его действия. Записывается в угловых скобках: `<p>`.

## П
**Параграф** — абзац текста. Создается тегом `<p>`. Браузер автоматически добавляет отступы до и после параграфа.

**Пустой элемент** — элемент, который не может содержать текста или других элементов (например, `<br>` или `<img>`). У них нет закрывающего тега.

## Р
**Разметка** — процесс расстановки тегов в тексте для придания ему структуры.

## С
**Семантическая верстка** — использование тегов по их смыслу (например, `<header>` для шапки сайта, `<nav>` для навигации, `<article>` для статьи), а не просто `<div>` для всего подряд.

**Ссылка** — см. Гиперссылка.

## Т
**Таблица** — способ представления данных в строках и столбцах. Создается тегами `<table>` (таблица), `<tr>` (строка), `<td>` (ячейка), `<th>` (заголовок).

**Тег** — основная команда HTML, заключенная в угловые скобки. Бывают открывающие (`<p>`) и закрывающие (`</p>`).

## У
**Уровень заголовка** — значимость заголовка от 1 (самый важный) до 6 (самый мелкий). На странице должен быть только один `<h1>`.

## Ф
**Форма** — элемент страницы для ввода данных пользователем. Создается тегом `<form>` и содержит поля ввода (`<input>`), кнопки, списки и т.д.

## Х
**HTML (HyperText Markup Language)** — язык гипертекстовой разметки, используемый для создания веб-страниц.

## Ц
**Цитата** — выделенный фрагмент текста, представляющий собой цитату. Для коротких цитат используется тег `<q>`, для длинных — `<blockquote>`.

## Ч
**Чекбокс** — элемент формы (флажок), который может находиться в состоянии "включено" или "выключено". Создается тегом `<input type="checkbox">`.

## Э
**Элемент** — основная структурная единица HTML-документа. Состоит из открывающего тега, содержимого и закрывающего тега (для непустых элементов).

**Эскейп-последовательность** — специальный код для отображения символов, которые используются в HTML (например, `&lt;` для `<` и `&gt;` для `>`).

## Я
**Якорь** — ссылка, ведущая на определенное место внутри текущей страницы. Создается с помощью атрибута `id` у элемента и ссылки вида `href="#идентификатор"`.

---

## Дополнительные термины (для выполнения нормы 25+)

**Адрес** — контактная информация автора страницы. Оборачивается в тег `<address>`.

**Глобальные атрибуты** — атрибуты, которые можно использовать с любым HTML-элементом (например, `id`, `class`, `style`, `title`).

**Заблокированный элемент** — элемент, который занимает всю доступную ширину и начинается с новой строки (например, `<p>`, `<div>`, заголовки).

**Инлайн-элемент** — строчный элемент, который располагается в строке и занимает только необходимую ширину (например, `<a>`, `<span>`, `<strong>`).

**Кодировка** — способ преобразования символов в байты. Для HTML5 рекомендуется UTF-8, указывается в `<head>`: `<meta charset="UTF-8">`.
