Lecture 03 — «Списки и ссылки»
Тема: Создание списков и гиперссылок
Цель урока: Научиться создавать маркированные и нумерованные списки, а также гиперссылки на другие страницы.
Время: ~20 мин.
Ключевые понятия: маркированный список, нумерованный список, элемент списка, гиперссылка, атрибут href

1) Введение
Списки помогают структурировать информацию: пункты меню, инструкции, перечни характеристик. Ссылки — это основа интернета, они позволяют переходить с одной страницы на другую.

2) Основная часть
Маркированный список (с точками) — тег <ul> (unordered list):

html
<ul>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ul>
Нумерованный список (с цифрами) — тег <ol> (ordered list):

html
<ol>
  <li>Шаг 1</li>
  <li>Шаг 2</li>
  <li>Шаг 3</li>
</ol>
Ссылки создаются тегом <a> (anchor):

html
<a href="https://example.com">Текст ссылки</a>
href — атрибут, указывающий адрес страницы

Между открывающим и закрывающим тегом пишется видимый текст ссылки

Внутренние ссылки (на страницы внутри вашего сайта):

html
<a href="page2.html">Перейти на вторую страницу</a>
3) Пример/практика
Создайте страницу "Рецепт". Оформите ингредиенты как маркированный список, а этапы приготовления — как нумерованный. Внизу добавьте ссылку на сайт с похожими рецептами.

Задание:

Создайте список покупок (маркированный)

Создайте инструкцию из 5 шагов (нумерованный)

Добавьте ссылку на любой сайт, открывающуюся в этой же вкладке
