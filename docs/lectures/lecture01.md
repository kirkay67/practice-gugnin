Lecture 01 — «Введение в HTML»
Тема: Основы языка гипертекстовой разметки
Цель урока: Понять структуру HTML-документа и научиться создавать простейшую веб-страницу.
Время: ~20 мин.
Ключевые понятия: HTML, тег, атрибут, структура документа, DOCTYPE

1) Введение
HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц. Браузеры читают HTML и отображают его в виде сайтов. HTML не является языком программирования — это язык разметки, который описывает, где находится заголовок, абзац, изображение или ссылка.

2) Основная часть
Любой HTML-документ имеет обязательную структуру:

<!DOCTYPE html> — сообщает браузеру, что это HTML5

<html> — корневой элемент страницы

<head> — служебная информация (заголовок вкладки, кодировка)

<body> — видимое содержимое страницы

HTML состоит из элементов, которые создаются с помощью тегов. Большинство тегов имеют открывающий и закрывающий вид:

<название_тега> — открывающий тег

</название_тега> — закрывающий тег

Пример минимальной страницы:

html
<!DOCTYPE html>
<html>
<head>
    <title>Моя первая страница</title>
</head>
<body>
    Привет, мир!
</body>
</html>
